# Web Wallets

A modern web application for managing cryptocurrency wallets on Ethereum and Solana blockchains. Built with React, TypeScript, and Tailwind CSS.

## Features

- **Multi-Chain Support**: Manage wallets for Ethereum and Solana.
- **Dashboard**: View balances, transactions, and portfolio overview.
- **Receive Funds**: Generate addresses and QR codes for receiving payments.
- **Settings**: Customize wallet preferences and security options.
- **Dark Mode**: Forced dark theme for optimal user experience.
- **Responsive Design**: Works seamlessly on desktop and mobile devices.

## Technologies Used

- **Frontend**: React, TypeScript, Tailwind CSS
- **Routing**: React Router
- **Icons**: Emoji-based icons for simplicity
- **Build Tool**: Vite (assumed based on typical setup)
